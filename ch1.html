Lesson 1.1: Introduction to HTML

HTML (Hyper Text Markup Language) is the standard markup language that is used for creating Web pages. It is the structure of a Web page and consists of elements. Elements are used as a way to tell the browser how to display content, things include a heading, paragraph, link, etc.

Example

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

Bài học 1.1: HTML là gì?

HTML (Hyper Text Markup Language) là ngôn ngữ đánh dấu tiêu chuẩn được dùng để tạo ra các trang web. Nó giống như là phần khung sườn của một trang web, được cấu thành từ các thành phần gọi là element. Mấy cái element này dùng để chỉ cho trình duyệt biết cách hiển thị nội dung ví dụ như tiêu đề, đoạn văn, liên kết, v.v.

Ví dụ

<!DOCTYPE html>
<html>
  <head>
    <title>Trang đầu tiên</title>
  </head>
  <body>
    <h1>Xin chào thế giới!</h1>
  </body>
</html>

Lesson 2.1: What is CSS?

CSS stands for Cascading Style Sheets. It’s the language used to style HTML content — for example, changing fonts, colors, layouts, and spacing.

What does “Cascading” mean?

It refers to the order in which rules are applied. If multiple rules affect the same element, CSS “cascades” them based on specificity and order.

Basic CSS Example

Here’s how to style a paragraph to be red and center-aligned:

p {
  color: red;
  text-align: center;
  }

This CSS can be placed inside a <style> tag in the HTML <head> section, or in a separate `.css` file linked with <link>.

Bài học 2.1: CSS là gì?

CSS là viết tắt của Cascading Style Sheets. Đây là ngôn ngữ dùng để tạo kiểu cho nội dung HTML — ví dụ như thay đổi phông chữ, màu sắc, bố cục và khoảng cách.

“Cascading” nghĩa là gì?

Nó đề cập đến thứ tự áp dụng các quy tắc. Nếu nhiều quy tắc áp dụng cho cùng một phần tử, CSS sẽ “xếp chồng” chúng theo mức độ ưu tiên và thứ tự xuất hiện.

Ví dụ CSS cơ bản

Đây là cách để làm cho đoạn văn bản có màu đỏ và căn giữa:

p {
  color: red;
  text-align: center;
  }

Đoạn CSS này có thể được đặt trong thẻ <style> trong phần <head> của HTML, hoặc trong một tệp `.css` riêng biệt được liên kết bằng <link>.
